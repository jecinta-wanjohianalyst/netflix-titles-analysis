# Netflix Titles Analysis

## Project Overview
This project explores the Netflix Titles dataset using Python and Jupyter Notebook. The objective was to clean the data, perform exploratory data analysis (EDA), and generate insights about Netflix's content library.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains information about movies and TV shows available on Netflix, including title, type, director, cast, country, release year, rating, duration, and genres.

## Tasks Performed
- Data cleaning and preprocessing
- Handling missing values
- Removing duplicate records
- Exploratory Data Analysis (EDA)
- Data visualization

## Key Insights
- Movies make up a larger share of Netflix's catalog than TV shows.
- Content production increased significantly over the years.
- The United States contributes the largest number of titles.
- Drama and International Movies are among the most common genres.
